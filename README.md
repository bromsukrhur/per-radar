# PER 레이더 — 저평가 성장주 대시보드

PC · 모바일 · 태블릿 완전 반응형 웹앱
한국투자증권 Open API 실시간 연동

---

## 빠른 시작

### 방법 1 — 브라우저에서 바로 열기 (데모 모드)
index.html을 더블클릭하거나 Chrome에서 열면 바로 실행됩니다.
데모 모드에서는 시뮬레이션 데이터로 모든 기능을 체험할 수 있습니다.

### 방법 2 — 실시간 API 연결 (권장)

1. Node.js 설치 (https://nodejs.org)

2. 이 폴더에서 터미널 실행:
   npm install express cors node-fetch
   node proxy.js

3. 브라우저에서 http://localhost:3001 접속

4. 우측 상단 "🔑 API 연결" 클릭 → App Key / Secret 입력

## 한국투자증권 API 발급

1. https://apiportal.koreainvestment.com 접속
2. 회원가입 후 로그인
3. [API 서비스] → [앱 등록] → 앱 이름 입력
4. App Key / App Secret 복사
5. 모의투자(Paper): 별도 계좌 불필요, 바로 사용 가능
6. 실계좌(Real): 한국투자증권 계좌 필요

## 기능

### 대시보드 탭
- 코스피/코스닥/환율/VIX 실시간 현황
- 업종별 PER 막대 차트 (반도체·방산·금융·전력기기·바이오·화장품)
- 코스피 선행 PER 히스토리 차트 (Chart.js)
- 고성장 저PER 종목 TOP 10 자동 정렬

### 종목 스크리너 탭
- 업종별 필터링, 이름/코드 검색
- 컬럼 클릭으로 정렬 (PER, PBR, ROE, EPS성장, 주가, 등락률)
- 종목별 투자점수 (1~5점 도트)
- 관심 종목 추가(⭐), 알림 설정(🔔) 원클릭

### 업종 분석 탭
- 6개 업종 전체 종목 상세 보기

### 알림 설정 탭
- PER 이하/이상, 주가 이하/이상, 등락률 조건 설정
- 브라우저 Push 알림 지원
- 알림 내역 실시간 표시
- 이메일·텔레그램 알림 (추후 확장 가능)

### 관심 종목 탭
- 저장된 관심 종목 모아보기
- localStorage에 자동 저장

## 주의사항
- API 키는 브라우저 localStorage에만 저장되며 외부로 전송되지 않습니다.
- 실계좌 API 호출 시 요금이 발생할 수 있습니다 (KIS 정책 확인).
- 본 앱은 정보 제공 목적이며, 투자 권유가 아닙니다.
- 모든 투자 판단과 책임은 사용자 본인에게 있습니다.
