# PER 레이더 — 저평가 성장주 대시보드 v2.0

PC · 아이패드 완전 반응형 웹앱
한국투자증권 Open API + 네이버 금융 + KRX 실시간 연동

---

## 빠른 시작

### 방법 1 — 브라우저에서 바로 열기 (데모 모드)
index.html을 더블클릭하거나 Chrome에서 열면 바로 실행됩니다.
데모 모드에서는 시뮬레이션 데이터로 모든 기능을 체험할 수 있습니다.

### 방법 2 — 실시간 API 연결 (권장)

1. Node.js 설치 (https://nodejs.org)

2. 이 폴더에서 터미널 실행:
   npm install express cors node-fetch
   node proxy.js

3. 브라우저에서 http://localhost:3001 접속

4. 프록시 서버가 실행되면 네이버 금융 PER/PBR 데이터가 자동으로 연동됩니다.

5. KIS 실시간 주가까지 원하면 우측 상단 "🔑 API 연결" 클릭 → App Key / Secret 입력

## v2.0 추가 기능

### 네이버 금융 실시간 PER (프록시 서버 실행 시 자동)
- 종목별 실제 PER, PBR, EPS 데이터
- 5분 캐시로 서버 부하 최소화
- KIS API 키 없어도 동작

### KRX 공식 업종 PER
- KRX 정보데이터시스템 공식 데이터
- 업종 평균 PER 실제값으로 업데이트
- 1시간 캐시

### KRX 저PER 전체 순위 탭 (신규)
- KOSPI 전체 종목 PER 0~15배 필터
- KRX 공식 데이터 기준 순위
- 관심 종목 바로 추가 가능

## 데이터 소스 우선순위

1. KIS API (실시간 주가) — KIS 키 필요
2. 네이버 금융 (PER/PBR/주가) — 프록시 서버만 있으면 됨
3. KRX 공식 데이터 (업종 PER, 저PER 순위) — 프록시 서버만 있으면 됨
4. 데모 데이터 — 아무것도 없어도 동작

## 아이패드에서 실데이터 보는 법

PC에서 proxy.js 실행 후 같은 와이파이에서:
1. PC의 IP 확인 (cmd → ipconfig → IPv4 주소)
2. 아이패드 브라우저에서 http://[PC의IP]:3001 접속

## 한국투자증권 API 발급 (선택사항 — 실시간 주가용)

## 기능

### 대시보드 탭
- 코스피/코스닥/환율/VIX 실시간 현황
- 업종별 PER 막대 차트 (반도체·방산·금융·전력기기·바이오·화장품)
- 코스피 선행 PER 히스토리 차트 (Chart.js)
- 고성장 저PER 종목 TOP 10 자동 정렬

### 종목 스크리너 탭
- 업종별 필터링, 이름/코드 검색
- 컬럼 클릭으로 정렬 (PER, PBR, ROE, EPS성장, 주가, 등락률)
- 종목별 투자점수 (1~5점 도트)
- 관심 종목 추가(⭐), 알림 설정(🔔) 원클릭

### 업종 분석 탭
- 6개 업종 전체 종목 상세 보기

### 알림 설정 탭
- PER 이하/이상, 주가 이하/이상, 등락률 조건 설정
- 브라우저 Push 알림 지원
- 알림 내역 실시간 표시
- 이메일·텔레그램 알림 (추후 확장 가능)

### 관심 종목 탭
- 저장된 관심 종목 모아보기
- localStorage에 자동 저장
